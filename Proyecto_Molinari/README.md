# Práctica 2 

# Ejercicio 10: Simulacion de competencia de cocina y ranking

Este proyecto procesa y analiza los puntajes de una competencia de cocina de 5 rondas, generando estadísticas individuales y un ranking final.

## Estructura del Proyecto
- `notebooks/`: Contiene el archivo principal `.ipynb` con la resolución de los ejercicios 1, 2 y 10.
- `src/`: Contiene el módulo `funciones.py` con las funciones de calculo.
- `README.md`: Instrucciones de uso del proyecto.

## Requisitos y Ejecución
1. Instalar **Python 3.x** y **Jupyter Notebook** (o la extensión de VS Code).
2. No se requieren librerías externas adicionales.
3. Abrir el archivo `notebooks/ejercicio_10.ipynb` y ejecutar todas las celdas en orden.
